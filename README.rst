===========================
Question2Answer Close 1.0b
===========================
-----------
Description
-----------
This is a plugin for **Question2Answer** that provides question closing functionality.

--------
Features
--------
- allows closing questions, which removes ability to answer, comment or vote on them.
- admin and mods can close questions by default
- optionally allow closing own questions
- adds notice below closed questions with linked name of closer and reason
- adds text to closed question titles
- all text fully customizable

------------
Installation
------------
#. Install Question2Answer_
#. Get the source code for this plugin from github_, either using git_, or downloading directly:

   - To download using git, install git and then type 
     ``git clone git://github.com/NoahY/q2a-close.git close``
     at the command prompt (on Linux, Windows is a bit different)
   - To download directly, go to the `project page`_ and click **Download**

#. navigate to your site, go to **Admin -> Plugins** on your q2a install and select the '**Enable question closing**' option, then '**Save**'.
